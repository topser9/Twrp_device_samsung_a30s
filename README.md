# TWRP Samsung Galaxy A30s

## How to compile it:

To build:
export ALLOW_MISSING_DEPENDENCIES=true

setup envirnoment
`. build/envsetup.sh`

then prepare
`lunch omni_a30sxx-eng`

make the image
`mka recoveryimage`

# How to find the image built

`cd /out/target/product/a30S`

see recovery.img
